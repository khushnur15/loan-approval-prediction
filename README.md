# 🏦 Loan Approval Prediction System

An AI-powered fintech web application that predicts loan eligibility using Machine Learning and provides explainable AI-based loan decisions through a modern banking interface.

## 🚀 Features

- 🔐 User Authentication (Login & Registration)
- 🤖 Machine Learning Loan Prediction
- 📊 AI Decision Report
- 📈 Risk Assessment Analysis
- 💰 Financial Profile Evaluation
- 🏦 Modern FinTech UI
- ⚡ Real-time Results

## 🛠 Tech Stack

### Frontend
- HTML5
- Tailwind CSS
- JavaScript

### Backend
- Flask
- Flask-CORS
- Flask-Login
- Flask-SQLAlchemy
- SQLite

### Machine Learning
- Scikit-Learn
- NumPy
- Joblib

## 📈 Model Information

- Model: Logistic Regression
- Accuracy: **80%**
- Prediction Type: Binary Classification
- Output: Approved / Rejected

## ⚙️ Run Locally

```bash
git clone https://github.com/khushnur15/loan-approval-prediction.git
cd loan-approval-prediction
pip install -r backend/requirements.txt
cd backend
python app.py
```

Open:

```text
http://127.0.0.1:5000
```
